&nbsp;查询城市&nbsp;:&nbsp;/api/findCityNameLike22?cityName=城市名称
&nbsp;查询城市&nbsp;:&nbsp;/api/findByCityNameLikeAndDescriptionIs?cityName=城市名称&description=描述
